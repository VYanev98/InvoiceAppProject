1. Before you start the project please configurate your database entry point and credentials. You must add them here - src\main\resources\application.properties.
2. If you use a different database, keep in mind that some dependencies may be missing from the pom file and need to be added.
3. When you start the application, the table is created automatically, but if you want to create it in advance, you can use the script from - Before You Start\CreateTabel.sql
4. You can find brief documentation of the project here - Before You Start\InvoiceAppProject_Documentation_BG.docx
